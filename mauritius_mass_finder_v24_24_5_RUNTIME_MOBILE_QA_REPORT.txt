CHANGES IMPLEMENTED
1. Sunday evening rule clarified and implemented.
   - “Sunday evening” / “messe dimanche soir” now means actual Sunday rows with time >= 17:00 only.
   - Saturday vigil rows are no longer included in this specific query intent.
   - Sunday obligation mode remains separate and still includes Sunday + Saturday from 15:30 onward.

2. Visible query cleanup improved after chip removal.
   - Removing “soir” from “messe du soir Curepipe” now leaves “Curepipe”.
   - Removing “matin” from “messe vendredi matin” now leaves “vendredi”.
   - Orphan connectors such as du/de/des/le/la/les/l/à/au/aux/d and generic words such as messe/church/église are removed from the displayed query after chip clearing.

3. Share payload is now bilingual.
   - App share text uses the active language.
   - Mass/church share text uses a localized intro label.
   - The previous hardcoded English subtitle in shareApp was removed.

4. Versioning updated.
   - app.js: 24.24.5
   - version.json: 24.24.5
   - manifest.json: 24.24.5
   - sw.js cache: mmf-v24-24-5
   - data/fallback metadata updated to v24.24.5

5. CSV/JSON schema harmonised.
   - coordinate_evidence_url is now present in JSON and fallback rows to match CSV.
