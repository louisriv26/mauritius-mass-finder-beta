Mauritius Mass Finder v24.17.6 Verification Report
Release: v24.17.6 Around-Time Semantic Fix

ROOT CAUSE
v24.17.5 treated approximate-time modifiers such as around/about/vers/environ/autour as after-time lower-bound intent. This changed user meaning and could exclude valid Masses before the target time, e.g. 17:30 for around 18:00.

CODE CHANGES
- Added parseTimeModifier(raw) with exact/around/after/before semantics.
- Changed parseAfterTime(raw) to return a lower-bound only for after/apres/après.
- Added parseBeforeTime(raw) for before/avant upper-bound intent.
- Added intent.timeModifier and intent.beforeMinutes.
- Updated rowMatchesIntent so exact/around uses ±60 minutes, after uses >= target, before uses <= target.
- Preserved exact-time-aware Near Me ranking for exact/around, while keeping after/before as range filters.
- Updated version/cache/release metadata to v24.17.6.
