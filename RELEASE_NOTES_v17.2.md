# Mauritius Mass Finder v17.2 — What’s New

v17.2 is a public-release hardening version focused on trust, clarity, bilingual consistency, mobile usability, update reliability, and Sunday/Vigil correctness.

## Main improvements

- Clearer badges for Sunday Mass, Saturday Vigil Mass, Weekday Mass, special rules, and other/ambiguous celebrations.
- Stronger bilingual trust wording reminding users to verify official sources for feast days, funerals, cyclones, holidays, and recent parish changes.
- Refined Help Center / User Guide text.
- Regression checks for search, including Notre-Dame, accent handling, hyphen handling, and Saint/St matching.
- PWA versioning updated to v17.2 with a new service-worker cache.
- Masses-only default preserved; other/ambiguous celebrations remain hidden unless explicitly enabled.
